<h4>Objetivo de software</h4>
<p>O sistema Scolaris será um sistema usado por colégios e instituições de
ensino para facilitar a comunicação entre professores e alunos, tendo funções
como: Mostrar aos alunos a suas notas trimestrais, e quanto falta para
completar a média para aprovação,professores poderão postar as avaliações,
juntamente com as notas individuais dos alunos em cada avaliação e um
recado caso seja necessário, é importante também destacar que o sistema
Scolaris não se limita a uma única sala de aula, podendo haver mais caso o
diretor julgue necessário.</p>

<h4>Funcionalidades do sistema</h4>
<h5> Criação e manutenção de turmas</h5>
<p>Ator:Diretor</p>
<p>Descrição: O diretor poderá criar e editar as turmas, escolhendo o
nome da turma, a descrição, os alunos que participarão e os professores,
podendo editar tais dados posteriormente caso necessário.</p>
<h5> Gerenciar avaliações </h5>
<p>Ator:Professor</p>
<p>Descrição: Os professores poderão adicionar avaliações, que terão
descrição, nota e data de aplicação, dessa mesma avaliação eles poderão
inserir as notas dos alunos, caso a avaliação seja excluída, todas as notas
também serão, o professor também poderá deixar um recado a cada prova
para o aluno, caso ache necessário.</p>
<h5> Verificar as notas</h5>
<p>Ator: Alunos</p>
<p>Descrição: Os alunos poderão consultar suas notas nas salas que
estão, vendo a pontuação que atingiram da pontuação total já registrada e
conferindo a nota restante necessária para ser aprovado, também poderão
ver o recado deixado na avaliação do professor para o aluno se houver.</p>
